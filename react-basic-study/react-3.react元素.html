1.react元素

通过jsx的学习已经知道,jsx经过编译之后最终生成的是 react 元素。而react元素是构成 React 应用的最小单位,它就是屏幕上看到的内容也就是最终的页面元素。 React DOM这个库就是负责更新 DOM 以便和 React 元素保持一致,跟vue类似React DOM 管理的也是称为“根” DOM 节点内的所有内容。也就是挂载点里所有的内容。 而想要将一个 React 元素渲染到根 DOM 节点中,只需把它们一起传入 ReactDOM.render()中即可。注意:这个函数只会调用一次。 注意:React 元素是不可变对象。一旦被创建就无法更改它的子元素或者属性。 React DOM 会将React元素和它的子元素与它们之前的状态进行比较,并只会进行必要的更新来使 DOM 达到预期的状态。 也就是说不是全部重新渲染的,而是比较react元素前后的不同,然后重新渲染实际改变了的内容。
