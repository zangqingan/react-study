react的使用跟vue类似也是分三步即可。学习时使用本地或者CDN链接(通过 CDN 的方式引入时需设置script标签的 crossorigin 属性这样可以有更好的错误处理体验)即可。 1.引入React核心库,react.development.js,react-dom.development.js(生产环境换成production.min.js)以及Babel编译器 babel.min.js以便使用JSX语法。 2.添加一个DOM容器,React组件会替换DOM容器内的任何已有内容 3.加载自己定义的 React 组件,在script标签中使用jsx 要添加 type="text/babel" 属性 React 应用程序的组成部分:元素和组件。
