;这种标签语法既不是字符串也不是 HTML,而是可以在js中书写html标签的语法就被称为:JSX,它是JavaScript 的语法扩展。 JSX 是用来生成 React “元素”,然后react会将这些元素渲染为页面中的真实DOM。 在 JSX 语法中,你可以在大括号 {//js表达式} 内放置任何有效的 JavaScript 表达式。 {} 类似vue的插值表达式,用来输出简单的js表达式/变量等。 注释:{/* 注释内容 */} 注意: 因为 JSX 语法上更接近 JavaScript 而不是 HTML,所以 React DOM 使用 camelCase(小驼峰命名)来定义属性的名称,而不使用 HTML 属性名称的命名约定。 同时跟vue类似也必须有一个根元素包裹。 例如,JSX 里的 class 变成了 className,而 tabindex 则变为 tabIndex 注意:也是只能有一个根元素,多个时使用 () 括号包裹。 本质:Babel 会把 JSX 转译成一个名为 React.createElement() 函数调用最终是生成虚拟dom。
